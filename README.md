# 🐍 Snake Game

A modular Snake game built in **Python** using **Object-Oriented Programming (OOP)** principles and the **Turtle graphics library**.

---

## 🚀 Features

- Smooth grid-based snake movement
- Random food spawning
- Real-time score tracking
- 💾 Persistent High Score saving (stored in a file)
- Wall collision detection
- Self-collision detection
- Clean modular OOP structure

---

## 🛠 Technologies Used

- Python 3.11
- Turtle Graphics
- Object-Oriented Programming (OOP)
- File Handling (for High Score persistence)

---

## 📂 Project Structure

snake_game/
│
├── main.py          # Game loop
├── snake.py         # Snake movement logic
├── food.py          # Food spawning logic
├── scoreboard.py    # Score system & high score logic
├── data.txt         # Stores highest score

---

## ▶️ How to Run

1. Clone the repository:

git clone https://github.com/datomumladzee/Snake-game.git

2. Navigate to the project folder:

cd Snake-game/snake_game

3. Run the game:

python main.py

---

## 🎮 Controls

↑ Arrow → Move Up  
↓ Arrow → Move Down  
← Arrow → Move Left
